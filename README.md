# 基于神经网络的AI对话程序

作者: x0r_fl0w

## 简介

ReKo是一个基于神经网络的AI对话程序，具有以下特点：
- 图形用户界面(GUI)
- 神经网络可视化推理进度显示
- 基于技术文档文件夹中的TXT文件进行词语预测
- 实时对话功能

## 功能特性

1. **GUI界面**：直观易用的图形界面
2. **神经网络模型**：使用LSTM网络进行文本预测
3. **可视化推理**：实时显示训练进度和损失函数变化
4. **文档学习**：可以从技术文档中学习并预测下一个词语
5. **对话功能**：支持人机对话交互

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

1. 点击"加载技术文档"按钮，选择包含TXT文件的文件夹
2. 点击"训练模型"按钮开始训练神经网络
3. 在输入框中输入文本，按回车或点击"发送"进行对话
4. 点击"预测下一个词"获取基于当前输入的词语预测结果

## 注意事项

- 确保技术文档文件夹中包含.txt格式的文本文件
- 训练时间取决于文档数量和计算机性能
- 第一次运行时需要安装依赖包
- 程序需要一定的计算资源，较低配置的计算机可能运行较慢
