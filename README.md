# ReKo AI - 基于统计方法的智能对话程序

作者: x0r_fl0w

## 简介

ReKo AI是一个基于统计方法的智能对话程序，具有以下特点：
- 图形用户界面(GUI)
- 可视化推理进度显示
- 基于技术文档文件夹中的TXT文件进行词语预测
- 实时对话功能
- 强化学习反馈机制

## 功能特性

1. **GUI界面**：直观易用的图形界面，支持中文显示
2. **文本分析**：使用n-gram统计方法进行文本预测
3. **可视化推理**：实时显示对话匹配度变化
4. **文档学习**：可以从技术文档中学习并预测下一个词语
5. **对话功能**：支持人机对话交互
6. **强化学习**：支持用户反馈评分，持续优化回复质量
7. **统计信息**：显示词汇量、词频统计等文档分析结果

## 技术架构

- **前端**：基于Tkinter的图形用户界面
- **核心引擎**：基于n-gram统计模型的文本分析器
- **强化学习**：基于Q-learning的对话策略优化
- **可视化**：使用matplotlib实时显示推理进度
- **配置管理**：YAML配置文件支持灵活参数调整

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

1. 运行程序：
   Windows：
   ```bash
   run.bat
   ```
   Linux：
   ```bash
   run.sh
   ```

2. 点击"加载技术文档"按钮，选择包含TXT文件的文件夹

3. 点击"处理文档"按钮开始构建统计模型

4. 在输入框中输入文本，按回车或点击"发送"进行对话

5. 点击"预测下一个词"获取基于当前输入的词语预测结果

6. 对AI回复进行评分（👍有用/👎没用）以帮助系统学习改进

## 配置说明

程序支持通过`config/config.yaml`文件进行配置，包括：
- 应用信息（名称、版本、描述）
- 界面设置（窗口大小、字体、主题）
- 分析参数（词汇量限制、n-gram阶数）
- 强化学习参数（学习率、折扣因子）
- 路径设置（文档、模型、日志路径）

## 注意事项

- 确保技术文档文件夹中包含.txt格式的文本文件
- 处理时间取决于文档数量和计算机性能
- 第一次运行时需要安装依赖包
- 程序需要一定的计算资源，较低配置的计算机可能运行较慢
- 支持中文文本处理，确保文档使用UTF-8编码
